## MedChat 远程问诊系统 - 启动说明

### 部署

将 `MedChat_v2.0.zip` 解压到任意目录即可使用，无需安装 Qt 环境。
解压后的文件夹结构：

```
MedChat_Package/
├── MedChat.exe        (主程序)
├── Qt5Core.dll        (Qt 运行时)
├── ...其他 DLL 和插件目录
└── history/           (首次运行后自动生成，存放聊天记录)
```

---

### 启动服务器

打开一个 **cmd 终端窗口**，进入程序所在目录，执行：

```
MedChat.exe --server
```

可选参数：

```
MedChat.exe -s -p 端口号
```

- `-s` 或 `--server`：以服务器模式启动
- `-p` 或 `--port`：指定监听端口，默认 9527

启动后终端会实时显示所有活动日志，包括用户上线/下线、消息转发（含内容预览）、文件传输、在线人数统计等。按 `Ctrl+C` 或关闭终端窗口即可停止服务器。

**注意**：服务器必须保持运行，客户端才能正常通信。

---

### 启动客户端

双击 `MedChat.exe`（不加任何参数）即可启动客户端。

- 默认连接 `127.0.0.1:9527`（本机服务器）
- 如需连接远程服务器，在登录界面点击「切换服务器」输入目标 IP 和端口

局域网内其他电脑的客户端只需将服务器地址改为运行服务器那台机器的局域网 IP 即可。

---

### 基本功能

**用户角色**：注册时选择「医生」或「患者」，联系人列表会显示对方角色。

**聊天**：点击左侧联系人进入对话，输入文字后点「发送」。消息以气泡形式展示，自己的消息在右侧（绿色），对方消息在左侧（白色）。

**文件传输**：对话中点击「发送文件」，对方会看到文件卡片，可选择接受或拒绝。

**文件传输助手**：联系人列表第一项「文件传输助手」，发消息会原样返回，发送文件会弹窗让你选择保存位置（相当于本地复制）。适合快速测试或自用文件中转。

**聊天记录**：切换联系人或关闭应用时自动保存当前对话，下次打开同一联系人会自动加载历史消息。记录存放在程序同目录的 `history/` 文件夹中。

---

### 典型使用流程

1. 在一台电脑上打开 cmd，运行 `MedChat.exe --server`
2. 双击 `MedChat.exe` 启动客户端，注册/登录
3. 在局域网内其他电脑上双击 `MedChat.exe`，切换到服务器 IP，注册/登录
4. 双方互相发消息、传文件即可
